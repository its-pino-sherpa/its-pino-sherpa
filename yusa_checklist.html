#2 나, 지금 유사연애
하고 있는 걸까?

체크리스트로 지금 내 관계를 확인해봐 · by 피노

연애는 귀찮은데, 연락할 사람은 있어야 해.
사귀지도 않고, 썸도 아닌데 매일 연락하는 그 관계.

설레는 건지, 그냥 허전한 건지 헷갈린다면
지금 이 체크리스트가 답을 줄 거야.

체크리스트 01

내가 유사연애를 하고 있다면?

해당되는 항목을 모두 선택해봐. 결과는 자동으로 나와.

✓
사귀고 싶은 마음은 없는데, 연락이 없으면 허전하다
✓
상대가 다른 사람과 가까워지면 기분이 나빠진다
✓
이 사람이 좋은 건지, 그냥 익숙한 건지 잘 모르겠다
✓
관계를 정의하고 싶지 않다. 지금이 편하니까.
✓
진지하게 만나자고 하면 부담스러울 것 같다
✓
이 사람 말고 더 좋은 사람이 나타나면 떠날 것 같다
✓
설렘보다는 안정감 때문에 연락하는 것 같다
✓
상대의 감정을 알면서도 적당히 거리를 유지하고 있다

0 / 8
체크리스트 02

내가 유사연애를 당하고 있나..?

상대방의 행동을 떠올리며 체크해봐.

✓
연락은 자주 하는데 만남은 항상 흐지부지된다
✓
새벽에 연락이 오거나, 심심할 때만 연락하는 것 같다
✓
내가 감정을 표현하면 갑자기 거리를 둔다
✓
관계를 정의하려 하면 "그냥 편하게 지내자"는 말이 돌아온다
✓
나는 설레는데, 상대는 늘 여유 있어 보인다
✓
대화는 주로 상대 얘기 위주고, 내 감정엔 별 반응이 없다
✓
내가 좀 무심하면 갑자기 연락이 오고, 반응하면 또 뜸해진다
✓
이 관계가 발전하고 있는 건지 아닌지 도무지 모르겠다
진단 결과
0 / 8
※ 이 체크리스트는 100% 정확한 심리 진단이 아니에요.
연애 상황은 사람마다, 맥락마다 달라서 점수가 전부를 말해주진 않습니다.
다만 지금 내 관계를 한 번쯤 돌아보는 계기가 됐으면 해요.
결국 나를 가장 잘 알아야 되는 건 나 자신이니까.
